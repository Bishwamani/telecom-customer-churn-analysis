# Telecom Customer Churn Analysis

This project analyzes customer churn in a telecommunications company using machine learning techniques such as clustering and Artificial Neural Networks (ANN).

## Project Objectives

- Identify patterns that influence customer churn
- Segment customers using clustering
- Predict customer churn using an Artificial Neural Network
- Provide business recommendations to improve customer retention

- ## Dataset

The dataset contains 7,043 customer records with features such as:
- Gender
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Churn status

- ## Project Workflow

1. Data Preparation
   - Handling missing values
   - Encoding categorical variables
   - Train-test split
   - Feature scaling

2. Clustering Analysis
   - KMeans clustering
   - Elbow method to determine optimal clusters
   - PCA visualization

3. Predictive Modeling
   - Artificial Neural Network (ANN)
   - Model evaluation using confusion matrix and ROC-AUC
  
   - ## Tools and Technologies

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab
- GitHub

- ## Results

- Customers were segmented into 4 clusters using KMeans.
- The ANN model achieved around 77% accuracy and 0.79 ROC-AUC score in predicting churn.
- Insights from clustering and prediction help identify high-risk customer groups.
